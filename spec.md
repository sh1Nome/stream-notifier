# 概要
* 配信の通知をするDiscordのBot

# 対応プラットフォーム
* YouTube
* Twitch

# 仕様
## アカウント登録
* `sn!regist [OPTION] [ACCOUNT]`
    * OPTION
        * `--youtube`
        * `--twitch`

## アカウント登録解除
* `sn!unregist [OPTION]`
    * OPTION
        * `--youtube`
        * `--twitch`

## 言語設定
* `sn!lang [OPTION]`
    * OPTION
        * `--ja`
        * `--en`

## 通知チャンネル設定
* `sn!notifyMe`
* `sn!notNotifyMe`

## ヘルプ
* `sn!help`
